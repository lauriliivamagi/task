# AGENTS.md

This project's guidance for AI coding agents and human contributors lives in
**[CLAUDE.md](CLAUDE.md)** — architecture, the development workflow, testing
utilities, coding standards (TigerStyle), and the feature/API reference.

For contribution setup and the release process, see
**[CONTRIBUTING.md](CONTRIBUTING.md)**. For decision history, see the ADRs in
**[docs/explanation/adr/](docs/explanation/adr/)**.

> `AGENTS.md` is the cross-tool convention for agent instructions; Task keeps
> the canonical content in `CLAUDE.md` and points here so any tool finds it.
> Keep a single source of truth — update `CLAUDE.md`, not this pointer.
